### 다운
> - 윈도우 버전
>> - https://ollama.com/download/windows
> - 리눅스 버전
>> - curl https://ollama.ai/install.sh | sh

### Model 설정
> - Model 다운 및 업데이트
>> - ollama pull (model)
> - Model 삭제
>> - ollama rm (model)
> - Model 복사
>> - ollama cp (model) (wanted_name)
> - 설치된 모델 리스트
>> - ollama list

### Model 실행
> - Model 실행
>> - ollama run (model)
