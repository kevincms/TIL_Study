### 정렬 종류
> 1. [버블 정렬](#bubble-sort)
>> - 인접한 데이터의 크기를 비교해 정렬
>> - 버블 정렬 중 한 loop에서 swap이 없으면 정렬 완료
> 2. [선택 정렬](#selection-sort)
>> - 최솟값 혹은 최댓값을 찾은 뒤 앞 혹은 뒤에서부터 정렬
>> - 역순 정렬에 최적의 효율
>> - 반대로 이미 정렬된 상태에서 자료가 추가되면 최악의 효율
> 3. 삽입 정렬
> 4. 퀙 정렬
> 5. 병합 정렬
> 6. 기수 정렬
